Aula Promogração

Comandos Terminal

  1. cd .. (volta 1 diretório)
  2. d: .. (troca de volume de dados)
  3. dir (mostra as pastas de arquivos)

Como apagar um hd pelo C.M.D (terminal)

  1. disckpart (entra no modo de formataçao de hd)
  2. list disck (abre a lista de unidades no pc)
  3. select disck (informa qual unidade voce quer usar)
  4. clean (limpa tudo q tem na unidade)
  5. Create partition primary (cria uma nova partiçao na unidade)
  6. format fs=nfts(formata a nova partiçao para uso)
  7. assign (para adicionar uma letra para a unidade)
  8. exit (para sair do terminal)

Comandos Github

  1. git clone url (clona um repositório do git)
  2. git pull (puxa as mudanças de código para o meu computador)
  3. git add . (adiciona tudo)ou (o nome do arquivo)
  4. git commit -m "seu comentario" (para informar o q foi feito)
  5. git push (envia os documentos para o github)

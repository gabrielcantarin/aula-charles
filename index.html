Aula Promogração

Comandos Terminal

  1. cd .. (volta 1 diretório)
  2. d: .. (troca de volume de dados)
  3. dir (mostra as pastas de arquivos)

Apagando hd

  1. disckpart (mostra todas as unidades no computado atualmente)
  2. select disck (informa qual unidade voce quer usar)
  3. clean (limpa tudo q tem na unidade)
  4. Create partition primary (cria uma nova partiçao na unidade)
  5. format fs=nfts(formata a nova partiçao para uso)
  6. assign (para adicionar uma letra para a unidade)
  7. exit (para sair do terminal)

Comandos Github

  1. git clone url (clona um repositório do git)
  2. git pull (puxa as mudanças de código para o meu computador)
  3. git add . (adiciona tudo)ou (o nome do arquivo)
  4. git commit -m "seu comentario" (para informar o q foi feito)
  5. git push (envia os documentos para o github)
